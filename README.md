Lab4: Сравнение рекурсивной и итеративной реализации факториала
Программа сравнивает время работы двух реализаций вычисления факториала: рекурсивной и итеративной (через цикл). Используется модуль timeit для замера времени и matplotlib для визуализации.

При запуске программа:

Выполняет бенчмарк для набора чисел от 10 до 150 (шаг 10)
Строит график зависимости времени от n
Выводит в консоль чистый бенчмарк одного вызова для n=50
Описание программы
Назначение
Сравнение эффективности двух подходов к вычислению факториала:

Рекурсивный (fact_recursive) — n! = n × (n-1)!, базовый случай 0! = 1
Итеративный (fact_iterative) — цикл от 1 до n с накоплением произведения
Алгоритмы
Рекурсивный факториал:

Базовый случай: при n = 0 возвращается 1
Рекурсия: n! = n × fact_recursive(n - 1)
Итеративный факториал:

Инициализация result = 1
Цикл for i от 1 до n: result *= i
Бенчмарк
Фиксированный набор данных: один список [10, 20, 30, ..., 150] для обоих методов
Усреднение: для каждого n выполняется 5 серий по 1000 вызовов, берётся минимальное время
Чистый бенчмарк: измерение времени одного вызова для n=50 (без усреднения)
Интерпретация графика
Ось X — размер входного числа n
Ось Y — время вычислений (секунды)
Итеративный метод обычно быстрее: нет накладных расходов на вызовы функций и стек
Рекурсивный метод при росте n увеличивает глубину стека, что замедляет выполнение
